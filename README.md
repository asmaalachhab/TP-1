# TP-1 : Projet « Temperature »

 Présentation
Ce projet a été développé dans le cadre d’un TP sur la gestion de températures.  
Nom du projet (dans IntelliJ IDEA) : **temperature**  
Dépôt GitHub : https://github.com/asmaalachhab/TP-1.git

 Objectif
- Mettre en œuvre une application Java (Swing) permettant de :  
  - Mesurer ou simuler des valeurs de température.  
  - Afficher (et éventuellement enregistrer) ces valeurs.  
  - Offrir une interface simple pour l’utilisateur.  
- Travailler avec le contrôle de version (Git) et le dépôt distant (GitHub).

 Structure du projet
- `src/` : Le code source Java (packages `com.projet.capteur.client`, `com.projet.capteur.serveur`, etc.).  
- `README.md` : Cette documentation.  
- `.gitignore` : Fichiers/dossiers ignorés par Git (ex : fichiers IntelliJ, .class, etc.).  
- Fichiers de configuration IntelliJ (`.idea/`, `.iml`, …).
